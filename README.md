# Social CRUD Application

## Project Description
This is a social-based application designed to demonstrate full CRUD (Create, Read, Update, and Delete) functionality. The app allows users to interact within a community by sharing content, managing their profiles, and engaging with others in a streamlined, user-friendly environment.

## Agile Planning

### User Stories
* **As a user,** I want to create a new post so I can share my thoughts with my network. (Create)
* **As a user,** I want to view a feed of all recent activities so I can stay updated. (Read)
* **As a user,** I want to edit my existing posts to fix mistakes or update information. (Update)
* **As a user,** I want to delete my posts or account to maintain control over my data. (Delete)

### Sprints
* **Sprint 1:** Repository setup, documentation, and environment configuration. (Current)
* **Sprint 2:** Implementation of User Authentication and Profile Creation.
* **Sprint 3:** Development of core Social Feed and Posting functionality.
* **Sprint 4:** Final testing, UI polishing, and deployment.
